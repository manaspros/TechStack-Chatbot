## **Getting Started**  
1. Clone the repository:  
   ```bash
   git clone https://github.com/your-username/techstack-gpt.git  
   cd techstack-gpt
   ```
2. Install dependencies:  
   ```bash
   npm install
   ```
3. Add your **Google Gemini API Key** to a `.env` file:  
   ```
   GEMINI_API_KEY=your-api-key
   ```
4. Start the server:  
   ```bash
   npm start
   ```
